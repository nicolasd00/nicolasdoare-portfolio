Projet 1 : Dashboard Utilisateurs Netflix

Ce tableau de bord visualise l’activité des utilisateurs de Netflix, notamment le nombre d’utilisateurs, les appareils utilisés, les abonnements sous différents angles d'analyse. La base de données utilisé provient d'un dataset Kaggle. Ce projet a été réalisé sur Tableau.
